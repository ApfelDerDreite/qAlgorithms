\doxysubsection{Detailed Description}
A class to store peak data. 

The \doxylink{classq_1_1_data_type_1_1_peak}{Peak} class is used to store peak data. It contains the position, height, width, and area of the peak. The class also contains the uncertainty of the peak position, height, width, and area, the data quality score of the peak, the regression coefficients, the valley position, the degrees of freedom, and the index of the regression window position.
