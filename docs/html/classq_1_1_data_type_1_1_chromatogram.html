◆ cuttingPoints

std::vector<std::unique_ptr<size_t> > q::DataType::Chromatogram::cuttingPoints

This vector stores information for cutting the chromatogram data.

The cuttingPoints vector can be seen as a job list for cutting the chromatogram data. Each element in the vector shows the number of data points counted from the end of the dataPoints vector that should be moved to the new object. The number of elements within the cuttingPoints vector determines the number of new objects that will be created. E.g. cuttingPoints = {10, 5} will create two new objects. The first object will contain the last 10 data points of the original object while these datapoints are removed from the original object. The second object will contain the next last 5 data points of the original object, while these datapoints are also removed from the original object. The original object will contain all data points except the ones that were moved to the new objects.
