\doxysubsection{Detailed Description}
A class to store LC-\/\+MS data. 

The \doxylink{classq_1_1_data_type_1_1_l_c___m_s}{LC\+\_\+\+MS} class is used to store LC-\/\+MS data. It contains the intensity, mz, ionization mode, measurement mode, MS level, retention time, and scan number of the LC-\/\+MS data. The \doxylink{classq_1_1_data_type_1_1_l_c___m_s}{LC\+\_\+\+MS} class is a subclass of the \doxylink{classq_1_1_data_type_1_1_mass_spectrum}{Mass\+Spectrum} class. \doxylink{classq_1_1_data_type_1_1_mass_spectrum}{Mass\+Spectrum}
