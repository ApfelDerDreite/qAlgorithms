Read LC-\/\+MS data from a CSV file. 

The read\+CSV method reads LC-\/\+MS data from a CSV file and stores it in the \doxylink{classq_1_1_l_c_m_s_data}{LCMSData} object. The method takes the filename of the CSV file, the start and end rows and columns, the separator, and the variable types. The variable types are used to determine the type of data in each column. The method requires the variable types\+: {\bfseries{scan\+Number}}, {\bfseries{retention\+Time}}, {\bfseries{mz}}, and {\bfseries{intensity}}.
