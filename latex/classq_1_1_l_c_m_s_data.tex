\doxysubsection{Detailed Description}
Derived class for handling LC-\/\+MS measurement data. 

This class implements the processing and management of LC-\/\+MS (Liquid Chromatography-\/\+Mass Spectrometry) measurement data. It extends the \doxylink{classq_1_1_measurement_data}{Measurement\+Data} class with specific attributes and methods required to handle LC-\/\+MS data, such as managing multiple intensity profiles corresponding to different retention times.
