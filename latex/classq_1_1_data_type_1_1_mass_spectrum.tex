\doxysubsection{Detailed Description}
{\bfseries{\doxylink{classq_1_1_data_type_1_1_mass_spectrum}{Mass\+Spectrum}\+:}} Class to represent a mass spectrum with additional metadata. 

This class stores an entire mass spectrum, including spectral data points, and metadata such as the MS level, ionization mode, and data type (profile or centroid).
